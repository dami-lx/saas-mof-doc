# 业绩总览卡片字段字典

## 1. 文档用途

这份字段字典用于统一 `业绩总览` 卡片的字段语义和业务概念。

配套文档：

- [业绩总览卡片客户业务说明](./业绩总览卡片-客户业务说明.md)
- [业绩总览卡片项目实现与排查说明](./业绩总览卡片-项目实现与排查说明.md)

## 2. 页面层字段

| 字段 / 概念 | 含义 | 备注 |
| --- | --- | --- |
| `performanceOverview` | 卡片类型标识 | 前端 config 名称 |
| `navPerformanceOverview` | 独立卡片 key | 前端 independent key |
| `收益走势` | 收益序列视图 | 关注累计收益和超额收益 |
| `净值走势` | 净值序列视图 | 关注单位净值、累计净值、调整净值 |
| `过滤异常点` | 对收益跳点做平滑处理 | 仅对收益走势曲线生效 |
| `包含转型前业绩` | 是否并入转型前历史 | 会改变可参与计算的历史 |
| `年化因子设置` | 自定义年化换算口径 | 影响部分年化指标 |
| `频率` | 原始 / 周 / 月 | 影响采样点和部分风险收益计算 |
| `日期区间` | 当前展示区间 | 预设或自定义 |

## 3. 顶层 DTO 字段

| 字段 | 含义 | 备注 |
| --- | --- | --- |
| `portfolio` | 组合侧指标集合 | 主体表现 |
| `benchmark` | 基准侧指标集合 | 报告基准表现 |
| `navTimeSeries` | 净值走势时间序列 | 常用于原始频率净值图 |
| `nav` | 兼容旧逻辑序列 | 部分老逻辑仍会使用 |
| `benchmarkTimeSeries` | 基准收益序列 | 供收益图对比使用 |
| `extReturnSeries` | 算术超额收益序列 | 前端标签为 `超额收益（算术）` |
| `activeReturnDivSeries` | 几何超额收益序列 | 前端标签为 `超额收益（几何）` |
| `originalNavFrequency` | 原始净值频率 | `day` / `week` / `month` |
| `monthlyNav` | 月频收益或净值序列 | 频率切换用 |
| `weeklyNav` | 周频收益或净值序列 | 频率切换用 |
| `monthlyNavTimeSeries` | 月频净值图序列 | 频率切换用 |
| `weeklyNavTimeSeries` | 周频净值图序列 | 频率切换用 |
| `transformDates` | 转型日期数组 | 供图上标记使用 |
| `isWeek` | 是否默认按周频展示 | 某些产品默认周频 |
| `canLoad` | 前端是否需继续补载 | 常见于首屏先占位 |

## 4. `portfolio` / `benchmark` 指标字段

| 字段 | 含义 | 备注 |
| --- | --- | --- |
| `annualTotalReturn` | 年化总收益 | 组合或基准自身年化收益 |
| `annualActiveReturn` | 年化主动收益 | 组合相对基准的年化超额收益 |
| `activeAnnualReturnDiv` | 年化主动收益（几何） | 几何口径超额收益 |
| `annualTotalRisk` | 年化总风险 | 年化波动率口径 |
| `annualActiveRisk` | 年化主动风险 | 相对基准超额收益序列的年化波动 |
| `totalReturn` | 累计收益 | 区间收益 |
| `totalRisk` | 区间总风险 | 非年化口径风险值 |
| `calmarRatio` | 卡玛比率 | 年化收益与最大回撤关系 |
| `sortinoRatio` | 索提诺比率 | 对下行风险更敏感 |
| `treynorRatio` | 特雷诺比率 | 单位系统性风险收益 |
| `hurst` | 赫斯特指数 | 趋势 / 均值回复特征 |
| `stutzer` | STUTZER 指数 | 偏离正态分布条件下的风险调整收益 |
| `realizedAlpha` | 实现阿尔法 | CAPM 语义下超额收益 |
| `ir` | 信息比率 | 单位主动风险对应的主动收益 |
| `sharpeRatio` | 夏普比率 | 单位总风险对应的超额收益 |
| `spearman` | Spearman 秩相关 | 相对报告基准 |
| `pearson` | Pearson 相关系数 | 相对报告基准 |
| `winRatio` | 投资胜率 | 相对基准胜率 |
| `profitLossRatio` | 平均损益比 | 相对基准平均损益比 |
| `accumulatedActiveReturn` | 累计主动收益 | 主动收益累计末值 |
| `accumulateReturnSeries` | 累计收益序列 | 用于图表绘制 |
| `maxDrawdownRecoverDays` | 最大回撤修复天数 | `-1` 常表示未修复 |

## 5. 持仓补充字段

| 字段 | 含义 | 备注 |
| --- | --- | --- |
| `accumulateAverageAssetValue` | 累计平均持仓市值 | 前端文案常称 `平均规模` |
| `accumulateAverageConcentration` | 累计平均持仓集中度 | 前端文案常称 `持券集中度` |
| `accumulateTurnoverRateYearly` | 累计平均持仓年化换手率 | 前端文案常称 `年化转手率` |
| `profitDaysRate` | 盈利天数占比 | 单期收益为正的期数占比 |
| `maxContinuousRisingDays` | 最大连涨天数 | 连续上涨最长区间 |

## 7. 前端参数字段

| 字段 | 含义 | 备注 |
| --- | --- | --- |
| `abnormalNavFilter` | 是否过滤异常净值点 | 仅收益走势曲线生效 |
| `customNavStartDate` | 自定义起始日期 | 图表区间 |
| `customNavEndDate` | 自定义结束日期 | 图表区间 |
| `transformationFundContinuation` | 是否包含转型前业绩 | 影响历史拼接 |
| `navSampleFrequency` | 净值采样频率 | `original` / `week` / `month` |
| `annualQ` | 年化因子参数 | 常见包含 `returnQDays` |
| `calendarType` | 日历类型 | 与年化因子联动 |

## 8. 重要展示规则

| 规则 | 含义 |
| --- | --- |
| 异常点过滤 | 只影响收益走势曲线，不影响其他结果 |
| 算术超额序列 | 来自 `extReturnSeries` |
| 几何超额序列 | 来自 `activeReturnDivSeries` |
| 默认频率 | 通常取 `originalNavFrequency`，部分产品可默认周频 |
| 转型标记 | 来自 `transformDates` |
| 持仓补充指标 | 只有在触发相关 reload 时才一定会加载 |

## 9. 基准侧兼容字段规则

| 字段 | 当前约定值 | 含义 |
| --- | --- | --- |
| `benchmark.winRatio` | `0D` | 基准侧不做相对自身胜率计算 |
| `benchmark.profitLossRatio` | `null` | 基准侧不做相对自身平均损益比 |
| `benchmark.spearman` | `1D` | 自相关兼容值 |
| `benchmark.pearson` | `1D` | 自相关兼容值 |

## 10. 最重要的记忆点

- `业绩总览` 是一个“指标 + 收益曲线 + 净值曲线 + 转型标记”的复合卡片。
- `abnormalNavFilter` 只改收益走势图，不改其他结果。
- `annualQ` 是年化口径修正，不是底层净值变了。
- `benchmark` 和 `portfolio` 不是完全对称对象。
- `extReturnSeries` 和 `activeReturnDivSeries` 不能混用。
