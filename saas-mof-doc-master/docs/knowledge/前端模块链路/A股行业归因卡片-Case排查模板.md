# A股行业归因卡片 Case 排查模板

## 1. 适用场景

适用于：

- 客户质疑行业归因结果
- 测试发现页面数字异常
- 行业卡片与 Brinson 总览不一致
- 新旧系统做行业归因比对

## 4. 最小排查顺序

1. 先确认当前是不是 `主动` 口径
2. 确认当前是 `年化` 还是 `非年化`
3. 确认行业分类是否一致
4. 确认是否拿行业卡片去和完整 Brinson 总览硬比较
5. 再决定要不要继续下钻公式

## 7. 按问题类型排查

### 7.1 页面无数据

检查项：

- 是否有 A 股持仓
- 是否有基准
- 是否形成完整归因周期
- 当前分类和穿透参数是否正常

### 7.2 总和与明细不一致

检查项：

- 当前是否同一行业分类
- 当前是否同一年化状态
- 是否用当前返回列表直接求和

记录模板：

```md
### 总和复核

- 收益明细求和：
- 页面收益总和：
- 风险明细求和：
- 页面风险总和：
- 权重明细求和：
- 页面权重总和：
```

### 7.3 行业卡片与 Brinson 总览不一致

检查项：

- 是否已确认行业卡片只展示 management
- 是否把 timing 也期待分摊到行业条目
- 是否比较的是同一年化口径

### 7.4 行业分类切换后结果差异大

检查项：

- 是否切换了行业分类体系
- 是否切换了行业层级
- 是否错误期待不同分类下结果一一对应

### 7.5 主动权重方向和收益方向不一致

检查项：

- 是否误把超配方向等同于收益方向
- 是否确认该行业本身收益和行业内选股收益方向

## 9. 最常见根因速查

| 现象 | 常见根因 |
| --- | --- |
| 行业卡片和 Brinson 树图对不上 | 行业卡片只看 management，不含 timing |
| 总和看起来不对 | 比较时年化状态或行业分类不一致 |
| 行业切换后数字变化大 | 分类体系切换本来就会改变结果 |
| 超配行业却没赚到钱 | 超配方向不等于收益方向 |
| 页面无数据 | 持仓、基准或归因周期不足 |
