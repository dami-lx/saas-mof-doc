# A股风格归因卡片字段字典

## 1. 文档用途

这份字段字典用于统一 `A股风格归因` 卡片的常用术语和字段语义。

适用场景：

- 前端联调
- 后端排查
- 测试验收
- 客户答疑前的口径确认
- 重构时建立新旧字段映射

## 2. 页面层字段

| 字段 / 概念 | 所在层 | 含义 | 备注 |
| --- | --- | --- | --- |
| `stockStyleAttr` | 前端 card key / independent key | `A股风格归因` 卡片唯一标识 | 前端和 `mom` 卡片体系用这个 key 关联 |
| `主动值` | 页面交互 | 查看组合相对基准的结果 | 对应后端 `style`，前端对象名为 `styleBenchmark` |
| `绝对值` | 页面交互 | 查看组合自身的结果 | 对应后端 `styleSelf` |
| `显示年化值` | 页面交互 | 切换年化 / 非年化展示口径 | 对暴露数值无影响 |
| `风格稳定性` | 页面展示 | 查看不同月份下收益 / 风险 / 暴露的分布 | 不是顶部累计图的简单重复 |
| `cne5` | 页面交互 | 风险模型版本 | 公共报告场景可切换 |
| `cne6` | 页面交互 | 风险模型版本 | 当前实现对行业分类有额外约束 |

## 3. 前端卡片数据结构

### 3.1 顶层对象

| 字段 | 含义 | 备注 |
| --- | --- | --- |
| `styleBenchmark` | 主动值数据对象 | 虽然名字里有 `Benchmark`，但它不是基准自身，而是主动值 |
| `styleSelf` | 绝对值数据对象 | 组合自身风格归因结果 |
| `custom.riskModelVersion` | 当前风险模型版本 | 页面切换后会触发 reload |

### 3.2 风格对象结构

每个风格对象通常包含：

| 字段 | 含义 | 备注 |
| --- | --- | --- |
| `latest` | 报告期汇总结果 | 用于顶部柱图 |
| `periods` | 分期结果 | 用于风格稳定性蛛网图 |

### 3.3 `latest` 结构

| 字段 | 含义 | 备注 |
| --- | --- | --- |
| `summaryStyleList` | 汇总后的风格列表 | 顶部三张柱图直接基于它展示和求和 |

### 3.4 `periods` 结构

| 字段 | 含义 | 备注 |
| --- | --- | --- |
| `periods[date]` | 某个日期下的风格汇总 | 日期一般形如 `YYYY-MM-DD` 或对应序列化日期 |
| `periods[date].summaryStyleList` | 某个月份点位的风格列表 | 用于蛛网图一圈数据 |

## 4. `summaryStyleList` 单项字段

### 4.1 核心展示字段

| 字段 | 展示名称 | 业务含义 | 是否受年化开关影响 |
| --- | --- | --- | --- |
| `name` | 因子名称 | 如 `行业`、`市值`、`价值`、`beta`、`特殊性成分` | 否 |
| `return` | 年化收益 | 年化口径收益贡献 | 是 |
| `deannualReturn` | 非年化收益 | 非年化口径收益贡献 | 是 |
| `risk` | 年化风险 | 年化口径风险贡献 | 是 |
| `deannualRisk` | 非年化风险 | 非年化口径风险贡献 | 是 |
| `weight` | 暴露 | 风格暴露值 | 否，和 `deannualWeight` 实际相同 |
| `deannualWeight` | 非年化暴露 | 风格暴露值 | 否，和 `weight` 实际相同 |

### 4.2 前端使用规则

| 页面状态 | 收益字段 | 风险字段 | 暴露字段 |
| --- | --- | --- | --- |
| 年化 | `return` | `risk` | `weight` |
| 非年化 | `deannualReturn` | `deannualRisk` | `deannualWeight` |

### 4.3 字段解释

| 字段 | 解释 |
| --- | --- |
| `return / deannualReturn` | 该因子在当前口径下的收益贡献 |
| `risk / deannualRisk` | 该因子在当前口径下的风险贡献 |
| `weight / deannualWeight` | 该因子的风格暴露 |

## 5. 常见名称的标准含义

| 名称 | 标准含义 | 展示特点 |
| --- | --- | --- |
| `行业` | 所有行业因子汇总后的统一展示项 | 不逐个行业展开 |
| `市值` | Size 相关风格因子 | 风险模型中文映射名 |
| `价值` | Value 相关风格因子 | 风险模型中文映射名 |
| `成长性` | Growth 相关风格因子 | 风险模型中文映射名 |
| `动量` | Momentum 相关风格因子 | 风险模型中文映射名 |
| `波动率` | Volatility 相关风格因子 | 风险模型中文映射名 |
| `流动性` | Liquidity 相关风格因子 | 风险模型中文映射名 |
| `杠杆率` | Leverage 相关风格因子 | 风险模型中文映射名 |
| `盈利收益率` | Earnings Yield 相关风格因子 | 风险模型中文映射名 |
| `beta` | 市场 beta 因子 | 风险模型中文映射名可能仍为英文 |
| `特殊性成分` | 无法由行业和风格因子解释的剩余项 | 只出现在收益 / 风险中，不出现在暴露图中 |

## 7. 底层算法概念字段

| 字段 / 符号 | 含义 | 业务解释 |
| --- | --- | --- |
| `w_p` | 组合权重 | 组合当前持仓权重 |
| `w_b` | 基准权重 | 基准当前持仓权重 |
| `w_a` | 主动权重 | `w_p - w_b` |
| `E` | 因子暴露矩阵 | 个股到风格因子的映射 |
| `f` | 因子收益向量 | 每个风格因子的收益 |
| `r` | 个股收益向量 | 股票实际收益 |
| `Σ_f` | 因子协方差矩阵 | 风险模型中的因子风险关系 |
| `s` | 个股特异风险 | 非系统性风险 |
| `X_p` | 组合暴露 | `w_p^T E` |
| `X_b` | 基准暴露 | `w_b^T E` |
| `X_a` | 主动暴露 | `X_p - X_b` |

## 8. 时间口径字段

| 字段 / 概念 | 含义 | 注意事项 |
| --- | --- | --- |
| `latest` | 报告期汇总结果 | 不是单日结果 |
| `periods` | 多期结果 | 用于稳定性图 |
| `t` | 权重日 / 风险日 | 风险和暴露取 `t` |
| `t+1` | 收益日 | 收益取 `t+1` |
| `YYYYMM` | 月聚合键 | 稳定性图会先按自然月聚合 |

## 9. 展示求和与过滤规则字典

| 场景 | 规则 |
| --- | --- |
| 顶部收益总和 | 当前 `summaryStyleList` 中收益字段直接求和 |
| 顶部风险总和 | 当前 `summaryStyleList` 中风险字段直接求和 |
| 顶部暴露总和 | 当前 `summaryStyleList` 中暴露字段直接求和 |
| 顶部暴露图 | 过滤掉 `特殊性成分` |
| 稳定性收益图 | 保留 `特殊性成分` |
| 稳定性风险图 | 保留 `特殊性成分` |
| 稳定性暴露图 | 过滤掉 `特殊性成分` |

## 10. 最重要的记忆点

- `styleBenchmark` 实际表示主动值，不是基准自身。
- `weight` 和 `deannualWeight` 在这张卡片里数值相同。
- `特殊性成分` 只进收益和风险，不进暴露。
- 顶部图看报告期汇总，稳定性图看按月聚合后的分布。
- 收益看 `t+1`，风险和暴露看 `t`。
