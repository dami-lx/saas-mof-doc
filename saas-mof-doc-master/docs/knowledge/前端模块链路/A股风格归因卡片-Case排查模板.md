# A股风格归因卡片 Case 排查模板

## 1. 文档用途

这份模板用于在出现“客户提问 / 测试异常 / 联调不一致 / 重构比对偏差”时，快速收集证据并统一排查路径。

建议原则：

- 先固化现场，再下结论
- 先确认口径，再怀疑公式
- 先确认主动 / 绝对、年化 / 非年化、顶部 / 稳定性三个维度

## 4. 第一步：确认问题属于哪一类

先把问题归到下面某一类：

```md
### 问题分类

- [ ] 页面无数据 / 报错
- [ ] 顶部收益异常
- [ ] 顶部风险异常
- [ ] 顶部暴露异常
- [ ] 顶部总和与明细不一致
- [ ] 稳定性图异常
- [ ] 主动值与绝对值理解不一致
- [ ] 年化 / 非年化切换理解不一致
- [ ] 与客户预期不一致但系统计算未必错误
- [ ] 与其他系统 / 新重构结果不一致
```

## 5. 第二步：先确认展示口径

这是最重要的一步。

```md
### 展示口径确认

- 当前查看的是：
  - [ ] 主动值
  - [ ] 绝对值
- 当前展示的是：
  - [ ] 年化
  - [ ] 非年化
- 当前问题发生在：
  - [ ] 顶部柱图
  - [ ] 风格稳定性蛛网图
- 是否涉及：
  - [ ] 收益
  - [ ] 风险
  - [ ] 暴露
- 是否误把 `特殊性成分` 当成暴露项：
  - [ ] 是
  - [ ] 否
```

很多误判都发生在这里。

## 6. 第三步：按问题类型排查

### 6.1 页面无数据

检查项：

- 报告期内是否有 A 股持仓
- 是否至少经历了一个完整归因周期
- 是否有基准信息
- 风险模型数据是否可用
- 当前卡片是否处于公共报告或特殊配置场景

建议记录：

```md
### 页面无数据检查

- A 股持仓是否存在：
- 报告期内持仓期点数：
- 基准是否存在：
- 风险模型版本：
- 前端错误提示：
- 后端是否返回空列表：
```

### 6.2 顶部总和与明细不一致

检查项：

- 当前是否看的是同一组 `summaryStyleList`
- 收益 / 风险是否按百分比换算后再看
- 暴露是否错误包含了 `特殊性成分`
- 是否把主动值和绝对值混看了

建议记录：

```md
### 顶部总和复核

- 当前对象：
  - [ ] styleBenchmark
  - [ ] styleSelf
- 当前字段：
  - [ ] return
  - [ ] deannualReturn
  - [ ] risk
  - [ ] deannualRisk
  - [ ] weight
  - [ ] deannualWeight
- 明细求和结果：
- 页面标题总和：
- 是否一致：
```

### 6.3 顶部暴露异常

检查项：

- 当前是主动暴露还是绝对暴露
- 是否误把暴露理解为收益
- 是否错误期待暴露随年化开关变化
- 是否把 `特殊性成分` 也算进去了

### 6.4 稳定性图异常

检查项：

- 问题发生在哪一张图：收益 / 风险 / 暴露
- 当前日期选择了哪些月份
- 是否理解成“月末点”，而非“同月平均”
- 是否把顶部累计结果拿来和稳定性月度结果直接比较

建议记录：

```md
### 稳定性图检查

- 当前日期选择：
- 是否最近 6 个月默认选择：
- 涉及图表：
  - [ ] 收益
  - [ ] 风险
  - [ ] 暴露
- 是否包含 `特殊性成分`：
- 是否确认该月为按自然月聚合结果：
```

### 6.5 收益和风险日期对不上

检查项：

- 是否已经确认 `收益看 t+1，风险 / 暴露看 t`
- 是否误把同一期收益和风险硬对齐

建议记录：

```md
### 日期口径检查

- 当前收益对应日期：
- 当前风险对应日期：
- 当前暴露对应日期：
- 是否已确认 `t` / `t+1` 规则：
```

## 9. 最小排查顺序

如果时间很紧，至少按下面顺序走：

1. 确认 `主动值 / 绝对值`
2. 确认 `年化 / 非年化`
3. 确认 `顶部 / 稳定性`
4. 确认是否涉及 `特殊性成分`
5. 确认是否涉及 `t / t+1`
6. 再决定要不要继续下钻算法

## 11. 常见根因速查

| 现象 | 常见根因 |
| --- | --- |
| 暴露和收益方向看起来矛盾 | 把暴露误当成收益 |
| 顶部与稳定性不一致 | 一个是汇总口径，一个是月度聚合口径 |
| 切年化开关后暴露不变 | 暴露本身不年化 |
| 暴露总和和明细不一致 | 误把 `特殊性成分` 算进去了 |
| 收益日期和风险日期不一致 | 收益看 `t+1`，风险和暴露看 `t` |
| 页面无数据 | 持仓不够、基准缺失、风险模型缺失或归因周期不完整 |
